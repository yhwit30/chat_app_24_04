## polling vs SSE(Server-Sent-Events)

- polling : 연결해서 요청을 보내면 서버에서 응답하고 연결을 끊음

- SSE(Server-Sent-Events) : 연결해서 요청을 보내고 서버가 응답을 해도 연결을 끊지 않음
